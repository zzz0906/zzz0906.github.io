HHKB's BS and Delete 按钮引起的疑惑

Posted on 2020-11-25 Edited on 2021-10-08

最近又再捣鼓捣鼓自己HKKB的键盘，一直没有整明白自己的HHKB后面那一串开关的用处，后面写了一串英文介绍也没弄懂是怎么使用。经过仔细探究发现主要要的三个用途。

我的是lite版本和上面图示稍又不同。我的SW4按键主要是用来区分alt和菱形按键的各起的作用。主要还是在Delete和BS键上面。经过反复研究和查阅资料发现，原来Delete按键指的是foward delete可能正常之前的计算机，装备了backspace （向后删除） 和 delete （向前删除）。根据维基百科的介绍。

{
The key is sometimes referred to as the “forward delete” key. This is because the backspace key also deletes characters, but to the left of the cursor. On many keyboards, such as most Apple keyboards, the key with the backspace function is also labelled “delete”.[1]
}

所以HHKB是为了遵循这个规则，当该按键向上打开时指的是向前删除，否则delete键指的是backspace。

冷知识+1。

终于弄清楚了HHKB调到delete为什么还是会向前删除并且无法后退的疑问。
